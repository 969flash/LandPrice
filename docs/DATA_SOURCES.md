## 1. 개별공시지가 — V-World Open API (주 수집 경로)

전국 × 전 연도 필지별 데이터를 자동화로 구축할 수 있는 **유일한 공식 경로**.

- **생산 기관**: 국토교통부 (국가공간정보센터). 원천: 국토정보시스템/K-Geo플랫폼 + 시군구 지자체
- **플랫폼 운영**: 공간정보산업진흥원 (V-World "디지털트윈국토")
- **엔드포인트**: `https://api.vworld.kr/ned/data/getIndvdLandPriceAttr`
  - 스펙: https://www.vworld.kr/dtna/dtna_apiSvcFc_s001.do?apiNum=25
  - data.go.kr 등재(연계형): https://www.data.go.kr/data/15124014/openapi.do
- **파라미터**: `pnu`(필수, **8자리 이상 접두어 허용** — 10자리 법정동코드를 넣으면 해당 법정동 전체 필지 반환), `stdrYear`(YYYY), `stdrMt`, `format`(xml/json), `numOfRows`(최대 1000), `pageNo`, `key`(필수), `domain`
- **응답 필드**: `pnu`, `ldCode`, `ldCodeNm`, `regstrSeCode`(대장구분), `mnnmSlno`(지번), `stdrYear`, `stdrMt`, **`pblntfPclnd`(공시지가, 원/㎡)**, `pblntfDe`(공시일자), `stdLandAt`(표준지여부), `lastUpdtDt`
- **응답 envelope**: `{"indvdLandPrices": {...}}` (2026-07-14 무키 호출로 실측 — `resultCode: INVALID_KEY` 반환 확인, 엔드포인트 가동 중)
- **트래픽**: 명목 일일 999,999,999건 (사실상 무제한; 초당 제한은 미문서화 → 파이프라인에서 보수적 속도 제한 적용)
- **인증키**: https://www.vworld.kr/dev/v4dv_apikey_s001.do
  - 개발키: 즉시 자동 발급, 유효기간 6개월 (연장 가능)
  - 운영키: 별도 심사 (~10일), 연 단위 갱신
- **⚠️ 미검증**: `stdrYear` 소급 하한 (1990 기대 — 원천 DB와 지자체 열람 서비스가 1990년부터 제공하므로). 키 발급 후 `scripts/probe_api.py`로 최우선 확인할 것.

### 참고: 폐기된 구(舊) 경로

- 구 data.go.kr API `apis.data.go.kr/1611000/nsdi/IndvdLandPriceService` — **2023-12-31 폐기** (404 확인)
- 국가공간정보포털(nsdi.go.kr) 전체 — **2023-12-31 서비스 종료** (전 도메인 무응답 확인). 데이터는 V-World로 이관
- 폐기 공지: https://www.data.go.kr/bbs/ntc/selectNotice.do?originId=NOTICE_0000000003431

## 2. 개별공시지가 — V-World 공간정보 다운로드 (최신 스냅샷 벌크)

최근 연도분을 빠르게 적재하는 보조 경로. **과거 전 기간은 제공하지 않음.**

- **URL**: https://www.vworld.kr/dtmk/dtmk_ntads_s002.do?svcCde=NA&dsId=6
- **구성**: 시도별 ZIP 17개 × {CSV(코드 AL_D151), SHP(코드 AL_D150, 연속지적도 결합)}
- **갱신**: 반기 (5월/11월 수집). 확인 시점 기준일 2026-05-26
- **내용 연도**: 최신 스냅샷만 배포. SHP는 기준연도 + 과년도 1~4 공시지가 컬럼 보유. 전국 CSV 총 행수 159,935,632 (data.go.kr 15052266 등재 기준) ≈ 필지당 약 4~5개 연도분으로 추정 (**정확한 연도 범위는 로그인 후 실측 필요**)
- **접근**: V-World 무료 회원 로그인 필수. 다운로드 엔드포인트: `POST /dtmk/downloadResourceFile.do?ds_id=20171128DS00144&fileNo=…` (CSV 기준; 세션 쿠키 필요)
- **CSV 스키마 (AL_D151, 13개 컬럼)** — 출처: 국가중점데이터 컬럼정의서(26.01.02, vworld.kr 배포):
  1. 고유번호(PNU 19자리) 2. 법정동코드(10) 3. 법정동명 4. 특수지구분코드 5. 특수지구분명 6. 지번 7. 기준연도 8. **기준월(01=1.1.공시분, 07=7.1.추가공시분)** 9. 공시지가(원/㎡) 10. 공시일자 11. 표준지여부 12. 데이터기준일자 13. 원천시도시군구코드

## 3. 개별공시지가 — 서울 열린데이터광장 (1990~2026 연도별 벌크, 검증 완료)

서울시 한정이지만 **전 기간 벌크가 무인증으로 열리는 유일한 광역 단위** — 부트스트랩 및 API 교차검증 기준 데이터로 사용.

- **데이터셋**: 서울시 개별공시지가 정보 (OA-1180)
- **URL**: https://data.seoul.go.kr/dataList/OA-1180/F/1/datasetView.do
- **파일**: `공시지가_1990년.zip` ~ `공시지가_2026년.zip` (37개, 연 4회 갱신)
- **다운로드 (실측 검증)**: `POST https://datafile.seoul.go.kr/bigfile/iot/inf/nio_download.do?useCache=false`, 폼 필드 `infId=OA-1180`, `infSeq=3`, `seq=<파일번호>` (1990→32 … 2026→92; 매핑은 페이지에서 자동 재스크레이핑)
- **스키마 (1990년 파일 실측)**: cp949 인코딩 CSV — `시도명, 시군구명, 법정동명, 토지코드(=PNU 19자리), 공시지가(원/㎡), 시군구코드, 법정동코드, 필지구분코드, 필지구분명, 본번, 부번, 기준년도, 기준년월`
- **특기**: 1990년 데이터도 **현행 행정구역 코드로 재키잉**되어 있음 (예: 1995년 신설 강북구 코드가 1990년 행에 등장) → 시계열 결합 시 현행 PNU 기준으로 정합
- **실측 결과 (1990·1991·2025 파일 전수 변환)**:
  - 행 수: 1990년 921,432 / 1991년 999,091 / 2025년 892,840 (서울 필지 규모와 정합)
  - **1월 기준분만 수록** — 기준년월이 전부 1월 (7.1 추가공시분 없음, 2025년 포함). 7월분은 V-World API(`stdrMt=07`)로만 보강 가능
  - 1990·1991년에 공시지가 0원 행이 각 1.2~1.5% 존재 (초기 전산화 시대 원천 특성; 원본 유지, QA에서 플래그)
  - PNU 변형 소수 존재: 17~18자리, 한글 부번 세분(예: `…13701가`) — 특수지 표기. 원문 그대로 보존

## 6. 보조 지표: 전국지가변동률조사 (월별 가격 신호)

공시지가는 제도상 연 1회(+변경 필지 7.1 추가)가 최대 밀도. 월 단위 신호가 필요하면 아래가 유일한 국가승인통계 (단, **필지 단위가 아닌 표본 지수**):

- **작성**: 한국부동산원 (국가승인통계 제315001호). 표본 80,000필지
- **공표 단위**: 전국/시도/252 시군구/3,136 읍면동(2015.7~), 용도지역·이용상황별
- **주기**: 월별(2005.1~), 그 이전 분기, 연간은 1975년까지 소급
- **접근**: R-ONE(https://www.reb.or.kr/r-one/), KOSIS(orgId=408), data.go.kr 파일(15112264)
- **활용**: 필지별 연간 공시지가에 시군구×용도지역 월별 변동률을 곱한 월 단위 보간 (실무 표준 기법; 별도 파생 데이터로 명확히 구분 저장)

## 7. 제도적 근거 (기간·밀도의 한계 규정)

- **근거법 계보**: 지가공시및토지등의평가에관한법률(1989 제정) → 부동산 가격공시 및 감정평가에 관한 법률(2005) → 현행 **부동산 가격공시에 관한 법률**(2016 전부개정) — https://www.law.go.kr/lsInfoP.do?lsId=001827
- **개별공시지가 최초 고시**: **1990-08-30** (당시 명칭 '개별토지가격', 국무총리훈령 제241호 근거; 1995년 법률 명문화)
- **공시 주기**: 매년 1.1 기준 → 5.31까지 결정·공시 (실무상 4월 말). 1.1~6.30 분할·합병·신규등록·지목변경 등 발생 토지는 **7.1 기준 → 10.31까지 추가 공시** (법 제10조, 시행령 제16·21조)
- **1990년 이전**: 4원화 체계(기준지가·토지시가·과세시가표준액·기준시가) — 필지 단위 공개 디지털 데이터 **없음** (국가기록원 해설: https://www.archives.go.kr/next/newsearch/listSubjectDescription.do?id=001389)
- **규모**: 전국 3,975만 필지 (2025년 지적통계, 2024.12.31 기준) 중 공시 대상 약 3,559만 필지 (2025년 기준; 국공유지 일부 등 제외)
